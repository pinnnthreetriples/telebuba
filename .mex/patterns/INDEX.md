# Pattern Index

Lookup table for all pattern files in this directory. Check here before starting any task — if a pattern exists, follow it.

| Pattern | Use when |
|---------|----------|
| [add-feature.md](add-feature.md) | Adding a new user-facing feature (UI in `features/` + business logic in `services/`) |
| [add-service.md](add-service.md) | Adding a new business-logic module in `services/<domain>.py` |
| [add-telegram-task.md](add-telegram-task.md) | Writing code that performs a Telegram action via the typed-action executor |
| [add-warming-job.md](add-warming-job.md) | Registering a new scheduled warming job on the shared APScheduler |
